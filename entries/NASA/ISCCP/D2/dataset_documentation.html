International Satellite Cloud Climatology Project (ISCCP) Version D2

This data was obtained from the NASA Langley Distributed Active Archive Center (DAAC). The NASA Langley DAAC is part of the Earth Observing System Data and Information System (EOSDIS). The EOSDIS gathers and processes data from NASA's Earth Science Enterprise (formerly, Mission to Planet Earth). Information about the Earth is stored in interconnected data centers, each of which specializes in a different discipline. The LaRC DAAC archives and distributes data relating to the Radiation Budget, Clouds, Aerosols, and Tropospheric Chemistry.
