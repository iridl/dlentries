                            3.  INTRODUCTION

3.1  Objective/Purpose.

     The 1 X 1 degree Greenness data set was compiled to calculate the energy 
     and water exchange between the biosphere and atmosphere with the simple 
     biosphere model (SiB).  With the SiB calculations the effect of 
     vegetation on climate can be incorporated in a General Circulation Model 
     (GCM) of the atmosphere.

3.2  Summary of Parameters.

     Greenness: Percentage green leaves of total leaves (green and dead)

3.3  Discussion.

     The greenness (GREENNSS) derived from the FASIR-NDVI product is used with 
     the land cover map (VEG_CLSS.VGC) and a set of live and dead leaf optical 
     properties assigned to each vegetation type to create within SiB2 
     weighted (by live / dead fraction) leaf optical properties for each grid 
     area for each month (Sellers et al., 1994, 1995).

                          4.  THEORY OF MEASUREMENTS

An estimate of the proportion of dead leaves within the canopy is provided by 
the method originally used by Dorman and Sellers (1989) for SiB1. It is 
assumed that when the canopy is growing, all the leaves are green. After the 
maximum leaf area is reached, and the canopy starts to lose green leaves, the 
greenness fraction is determined by assuming that dead leaves remain within 
the canopy for one month before falling off or being eaten.

                                   6.  PROCEDURE

6.1  Data Acquisition Methods.

     The 1 X 1 degree Greenness data set is calculated from the 1 X 1 degree 
     global FPAR and VEG_CLSS.VGC data set.

6.2  Spatial Characteristics. 

     6.2.1  Spatial Coverage. 

            The coverage is global.  Data in each file are ordered from North 
            to South and from West to East begining at the international 
            dateline.  Point (1,1) represents the grid cell centered at 89.5 N 
            and 179.5 W (see section 8.4).

     6.2.2  Spatial Resolution. 

            The data are given in an equal-angle grid that has a spatial 
            resolution of 1 X 1 degree lat/long.

6.3  Temporal Characteristics. 

     6.3.1  Temporal Coverage.

            January 1987 through December 1988.

     6.3.2  Temporal Resolution. 

            Monthly.

9.2  Data Processing Sequence

     9.2.1  Processing Steps and Data Sets

            Two data sets and 1 lookup table (see below) are used for the 
            calculation of Greenness.

            1. 1 X 1 degree FPAR data set.
            2. VEG_CLSS.VGC data

            C CALCULATE LEAF AREA INDEX AND GREENNESS FRACTION FROM FPAR.
            C
            C FPARM = FPAR previous month
            C BARKD, STEMSD = biophysical, vegetation type dependent variables
            C
            C EXPONENTIAL EXTINCTION USED FOR BROADLEAVES : 1,2,6,7,8,10,11,12
            C LINEAR MODEL ( CLUMPED CANOPIES ) USED FOR  : 4,5,9
            C MIXTURE OF LINEAR AND EXPONENTIAL MODELS    : 3
            C
            C-----------------------------------------------------------------
            C
            C
            C
               RLINE = 0.
               IF( IV .EQ. 4 .OR. IV .EQ. 5 .OR. IV .EQ. 9 ) RLINE = 1.
               IF( IV .EQ. 3 ) RLINE = 0.5
            C
               LAI1  = -1.0/BARK(IV) * ALOG(1.-FPAR ) * (1.-RLINE)
              &       + FPAR  * LAIMAX(IV) * RLINE
               LAI0  = -1.0/BARK(IV) * ALOG(1.-FPARM) * (1.-RLINE)
              &       + FPARM * LAIMAX(IV) * RLINE
               DEAD  = AMAX1( 0.0001,( LAI0-LAI1) ) + STEM(IV)
               GREEN = LAI1/( LAI1 + DEAD )
               LAI   = LAI1 + DEAD
            C
               RETURN
               END

            where
               BARK   = -1/LAIMAX(IV)*ALOG(0.05)
               LAIMAX = maximum LAI for particular vegetation/land cover type
               IV refers to land cover class
               LAI0 = GREEN LAI of previous month
               LAI1 = GREEN LAI of current month
               LAI  = total LAI (live + dead material)

             IV refers to land cover class, see documentation VEG_CLSS.VGC.

                                   10.  ERRORS

10.1  Sources of Error.

      1. Errors inherited from other data sets (FASIR, FPAR, assumptions in 
         calculation of LAI).
      2. assumptions underlying calculation (e.g. that dead leaves remain on 
         vegetation for 1 month).
      3. fixed value for the STEM parameter from the lookup-table (9.2.1).

10.2  Quality Assessment. 

      10.2.1  Data Validation by Source.

              A rigorous verification of GREENNESS has still to be made 
              (Sellers et al., 1994, 1995).

      10.2.2  Confidence Level/Accuracy Judgment. 

              Greenness is not thought to be very accurate.

      10.2.3  Measurement Error for Parameters and Variables
