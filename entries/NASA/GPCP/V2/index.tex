There's a version 2 of the NASA GPCP precipitation data that starts 
earlier than version 1 and is being updated. Steve and Tony thought it 
would be a reasonable addition to the Data Library.

The binary data and description is found here:
http://precip.gsfc.nasa.gov/gpcp_v2_data.html

More info is at http://precip.gsfc.nasa.gov/ and 
http://precip.gsfc.nasa.gov/gpcp_v2_comb.html
