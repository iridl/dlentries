Hi Benno,

We have two new sets of retrospective fcst data ready to be posted on the 
Data Library.  They are echam4.5 AGCM retro runs forced with the ECMWF SST 
hindcast data sets. 

The data period is 1958-2001, for only 4 start months (Feb, May, Aug, Nov) 
and 0-4 months (5) lead.  Included below is an example control file of the 
monthly data for Nov.

Each set of the retro runs has 24 ensembles: the 1st set for SST scenarios
and 2nd set for ensemble-mean SSTs.
(for 2nd set, the 1st line of the .ctl file is changed to:
 /Data/data1/iri/forecast/shuhua/retro_ecem.D/Retro_em01.D/nov/echamcout.%mc%y4)

Please let me kniow if any additional information is required.

Thanks,
Shuhua

--
