datasetdocumentationheader Users of this data are asked to include in their reference section the following reference: Roeckner, E., and Co-Authors, 2006: Sensitivity of simulated climate to horizontal and vertical resolution in the ECHAM5 Atmosphere Model. J. Climate, 19, 3771-3791.

In the acknowledgment section the following information should be included: The ECHAM5 AGCM has been developed by the Max-Planck Institute for Meteorology and the code kindly provided to IRI for preforming these experiments. Technical help in setting up the model has been supplied by Luis Kornblueh, Uwe Shulzweida, and Uli Schlese from Max-Planck. The integrations performed here were completed using a computing grant from the National Center for Atmospheric Research Computing Simulation Laboratory program to IRI.

If you have questions about this data set please contact Dave DeWitt at daved@iri.columbia.edu.
