Description of the ECMWF ERA-Interim reanalysis dataset

Documentation for this data set can be found at the ECMWF ERA-Interim page.

Because of the conditions of use imposed by ECMWF, these data should only be used for research purposes within the IRI. Before you use these data, please register with ECMWF and read and agree to the ECMWF archive Conditions of Use.
