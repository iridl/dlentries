Description of CPC Brazil Daily Precipitation Analysis (retrospective, 1 January 1948 - 31 July 2005, 2005 Version)

The following information is located in the README file found at the ftp site ftp.cpc.ncep.noaa.gov/precip/wd52ws/brazil : 


About the Analysis:

1. Data Source: CMCD/INPE, INMET, FUNCEME/CE, LMRS/PB, EMPARN/RN, DMRH/PE, 
                DHME/PI, CEPES/SE, NMRH/AL, SRH/BA, CEMIG-SIMGE/MG, SEAG/ES, 
		SIMEPAR/PR, CLIMERH/SC.


2. Resolution:  1.0 degree x 1.0 degree (lon,lat)


3. Spatial Domain: 80W - 30W; 40S - 10N


4. Time Domain:  January 1, 1948 - December 31,2005 


5. Analysis Scheme: Modified Cressman (1959) Scheme (Glahn et al. 1985; Charba
                    et al. 1992). 


6. Quality Control: A. Duplicate Station Check
                    B. Data Check ( buddy check )

7. Format:   Sequential access binary files created on a little_endian platform.
             Missing value is -999.00.

Note 1. This data set will be updated and extended (in time) when new station 
data are available to us.

References:

Silva, V. B. S., V. E. Kousky, W. Shi, and R. W. Higgins, 2006: An improved gridded historical daily precipitation analysis for Brazil. Submitted to the Journal of Hydrometeorology.

Please see the following pages at the NCEP Climate Prediction Center for more information: Retrospective Analyses, Daily Precipitation Analysis.
