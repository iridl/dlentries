Description of CPC Brazil Daily Precipitation Analysis (retrospective, 1960-1997)

The following information is located in the README file found at the ftp site ftpprd.ncep.noaa.gov/pub/precip/wd52ws/brazil : 


About the Analysis:

1. Data Source: CMCD/INPE, INMET, FUNCEME/CE, LMRS/PB, EMPARN/RN, DMRH/PE, 
                DHME/PI, CEPES/SE, NMRH/AL, SRH/BA, CEMIG-SIMGE/MG, SEAG/ES, 
		SIMEPAR/PR, CLIMERH/SC.


2. Resolution:  1.0 degree x 1.0 degree (lon,lat)


3. Spatial Domain: 80W - 30W; 40S - 10N


4. Time Domain:  January 1, 1960 - December 31, 1997


5. Analysis Scheme: Modified Cressman (1959) Scheme (Glahn et al. 1985; Charba
                    et al. 1992). 


6. Quality Control: A. Duplicate Station Check
                    B. Data Check ( buddy check )


This data set will be updated and extended (in time) when new station data are 
available to us.

Please see the following pages at the NCEP Climate Prediction Center for more information: Retrospective Analyses, Daily Precipitation Analysis.
