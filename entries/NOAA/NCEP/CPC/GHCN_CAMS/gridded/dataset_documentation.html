CPC Monthly Global Surface Air Temperature Data Set Documentation

The following information is from the "README.txt" file for this data set found at ftp://ftp.cpc.ncep.noaa.gov/wd51yf/GHCN_CAMS/. 


         CPC Monthly Global Surface Air Temperature Data Set 
                      at 0.5 degree from 1948-present

1. Description of the data

   The unit for the temperature is (K), averaged monthly in GRIB format. 

   The data sets will be routinely updated around 8th of each month. 

2. Raw spatical resolution & Coverage:

   0.5 deg lat x 0.5 deg lon;
  -89.75S -- 89.75N;
   0.25E -> Eastward -> 0.25W (or 0.25E -> 359.75E);

   from January 1948 to present;

   The data set is also availabe at T126, T62 and 2.5x2.5 resolutions 
   in this directory.
                                                                                
3. Missing values:

   9.999E+20

-----------------------------------------------------------------------------
4. Reading and using the data
   A. GrADS data control files (___.ctl) in the same directory. 
   B. Use wgrib to manipulate, inventory, and decode GRIB files. See:
      	http://www.cpc.noaa.gov/products/wesley/wgrib.html.
-----------------------------------------------------------------------------
