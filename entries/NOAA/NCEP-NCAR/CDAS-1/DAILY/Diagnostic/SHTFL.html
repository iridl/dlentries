The original reanalysis is supplied by year and updated periodically. This table shows when the datafiles were last updated. Think of these as version labels, i.e. if the time has changed since you last looked at the data, the data may have changed. This is because new dates reflect NOAA file times -- the actual copying may occur a few days to a few weeks later.

The current year is updated daily and trails a few days behind the calendar date.
