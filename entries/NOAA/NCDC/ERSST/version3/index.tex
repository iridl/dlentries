As of 8 Apr 2010, the dataset was restored at NCDC with string dates.

The old date conversion was

time
dup monthlyedgesgrid
   second last subgrid
   partialgrid monthtimegrid replaceGRID


the new one is

 time
   /units (ISO8601) def
   (days since 1960-01-01) gridunitconvert
  replacefirstofmonthGRID
