This analysis product is based on all stations, near real-time and non real-time, in the GPCC data base supplying data for the individual month. The GPCC Full Data Monthly Product Version 2018 covers the period from 1891 to 2016 ; this new extended product version using the new GPCC climatology as analysis background was generated in May 2018. The data coverage per month varies from less than 10,000 to more than 50,000 stations.
