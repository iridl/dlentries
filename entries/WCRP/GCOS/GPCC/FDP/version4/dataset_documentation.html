WCRP, GCOS, GPCC Full Data Product, Version 4

IMPORTANT NOTES FROM THE GPCC FULL DATA PRODUCT VERSION 4 DOWNLOAD PAGE:

Note !
The archives 01.1921-12.1930, 01.1931-12.1940, 01.1951-12.1960, 01.1971-12.1980, 01.1981-12.1990, 01.2001-12.2007 have been updated on July 10th 2008 for 0.5°, 1.0° and 2.5°.
This was required because of corrupted data for the months: 10.1927, 05.1928, 11.1932, 01.1933, 12.1953, 08.1974, 07.1987, 10.2005, 11.2005, 12.2005, 01.2006 and 02.2006.

Important Note ! 07/23/2008
Due to data problems detected in some regions and month of the period 1901-2007 a revised version of the Full Data Product will be generated by us within the next weeks !!
Please await the publication of the revised version !!

In the IRI Data Library copy of this dataset, available here, from the initial time that this dataset was available in the IRI Data Library, beginning on 17 July 2008 until 3 October 2008, the July 10th 2008 corrections referred to in the first note above were already included. But during this time the IRI Data Library version of the dataset included the data problems noted in the 23 July 2008 note above.

On 3 October 2008, the IRI Data Library copy of this dataset was updated with new files generated at GPCC on 25 September 2008.

On 3 November 2008, the IRI Data Library copies of the 0.5° resolution Full Data Product Version 4 data files for the years 1911 to 1920 were replaced with new files generated at GPCC on 15 October 2008. This update corrected errors that affected data for the period March-May 1914.

On 1 April 2010, the IRI Data Library copies of the 0.5, 1.0 & 2.5° resolution Full Data Product Version 4 data files for September 2004 were replaced with new files generated at GPCC as of 9 October 2009. According to GPCC, this update corrected for CLIMAT value coding errors for some stations.
