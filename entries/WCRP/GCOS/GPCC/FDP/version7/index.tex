 /summary (The Full Data Reanalysis Product is of much higher accuracy compared to the GPCC near real-time products mentioned above. Therefore, its application is recommended for hydrometeorological model verification and water cycle studies, e.g. in context of UNESCO, GEWEX, and GTN-H (Global Terrestrial Network for Hydrology). This analysis product is based on all stations, near real-time and non real-time, in the GPCC data base supplying data for the individual month. The GPCC Full Data Reanalysis Product Version 7 covers the period from 1901 to 2013 ; this new extended product version using the new GPCC climatology as analysis background was generated in Apr. 2015. The data coverage per month varies from less than 10,000 to more than 48,000 stations. The full data re-analyses will be updated at irregular time intervals subsequent to significant data base improvements.) def
