Water Requirement Satisfaction Index (WRSI)

The WRSI dataset can be found here. It includes the Current WRSI product valid at the end of each growing season during 1961-1996. More recent data is available from the Africa Data Dissemination Service (ADDS).

Users are incouraged to read and reference the complete documentation for the WRSI and CHARM products, which has been made available by the United States Geological Survey (USGS).
