Getting permission to download CRU TS3.23 data

Distribution of the CRU TS3.23 data set is handled by the British Atmospheric Data Centre (BADC). To access the TS3.23 data set from BADC directly, one must register as a data user at BADC (see https://services.ceda.ac.uk/cedasite/register/info/).

With the permission of BADC, we have established a mechanism to allow users to download TS3.23 data via the IRI Data Library. When requesting a data download, you will be asked to agree to the Terms & Conditions before proceeding.
