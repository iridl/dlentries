Getting permission to access CRU TS3.21

Distribution of the CRU TS3.21 data set is handled by the British Atmospheric Data Centre (BADC). To access the TS3.21 data set from BADC directly, one must register as a data user at BADC (see http://badc.nerc.ac.uk/reg/user_register_info.html).

With the permission of BADC, we have established a mechanism to allow users to download TS3.21 data via the IRI Data Library. When requesting a data download, you will be asked to agree to the Terms & Conditions before proceeding.
