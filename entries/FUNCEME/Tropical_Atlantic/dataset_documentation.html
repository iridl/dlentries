Tropical Atlantic SST and Wind Pseudo-stress data from FUNCEME

This data set contains gridded (2° lon/lat) monthly sea surface temperature (SST) and wind pseudo-stress data for the tropical Atlantic Ocean, including monthly means and anomalies (from the 1964-2003 base period). The monthly climatology itself (1964-2003 base period) is also available. The data can be found here. Now fully processed at FUNCEME, this data set is an update of Jacques Servain's tropical Atlantic SST and wind stress data set.

The preparation of the data is described on the following page from FUNCEME: http://www.funceme.br/DEMET/pirata/metodo.htm. The formatting of the original data is described here: http://www.funceme.br/DEMET/pirata/formato.htm
