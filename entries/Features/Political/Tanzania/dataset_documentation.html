2002 Shapefiles for Tanzania

Source: Judy Omumbo, IRI (20070612)

ILRI have recently digitized Tanzania boundaries (work completed in 2006) based on the last census (2002).
This file is available on their web site at http://www.ilri.org/gis/search.asp?id=442. Wards are equivalent to Admin4.
