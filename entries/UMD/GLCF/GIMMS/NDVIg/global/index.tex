This data set was updated through the end of 2006 on 06 Aug 2010.  The new files are "shortarraytype"
rather than "realarraytype", and the spatial domain is a bit different -- including a difference between
the NDVI and flag values.  The new data are not described as a new version -- the old data have simply
been replaced at GLCF/GIMMS.  The description file for the previous installment of this data set, which
ran through 2003, is given below, and the data files remain on /Data/data7.  The data for the updated
version are in a new subdirectory called "update2006".
