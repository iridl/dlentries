The weekly subdataset is the same data as the data here. Only for some reason the variables here were set on a daily S grid even though there are files only on a weekly basis. Thus, the weekly subsdataset is set on a weekly S grid and should be more performant to use as it is 7 times smaller without as many missing values. The previous version of the daily variables is preserved here for reproducibility purposes. New users are encouraged to use the weekly version.
