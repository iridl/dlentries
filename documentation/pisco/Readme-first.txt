====================
WHAT IS PISCOp?
====================

PISCOp is a new local historic gridded rainfall dataset (1981–present) with a daily temporal and 0.1° spatial resolution.
PISCOp has two versions:
	 - A stable version (1981-2016) that present a visual and automatic quality control.
	 - and, A unstable version (1981-present) that present only an automatic quality control, the rasters for the period 1981-present correspond to the stable version.
