DOE Precipitation and Temperature Anomalies

Dataset identifier:DOE

One-line description: DOE Gridded Surface Precipitation and Temperature Anomalies

Detailed description: This global precipitation and temperature anomaly data set for the period 1851 through 1990 has been assembled under funding by the U.S. Department of Energy.

NOTE: The netCDF data in this directory are not yet completely compliant with the CDC netCDF Standard Format. The differences are very slight, but should be noted. We plan to modify the data files to be strictly compliant with the CDC netCDF Standard Format by May 1994.

Archive parameters: File names are composed of abbreviated designations for the variable (air or precip) and time period (ann, seas, or mon): 

(variable).(time).nc
Variables and Units: 
 
Temperature data is given in deg C.
Precipitation data is given in mm.
These data are in the process of being cataloged here.

Precision: Grid point anomalies have been calculated to an accuracy of .01 deg C for temperature data. This does not necessarily reflect the accuracy of the original data, however. Individual monthly grid point anomalies are probably accurate to .2 deg C. Precipitation data includes a precision of .01 mm for monthly data and .001 for seasonal data. The individual monthly grid point anomalies are probably accurate to 10 mm.

Spatial coverage:

    90N-90S (by 5.0), 0-355E (by 5.0)
Temporal coverage:
  • 1851 through 1990
  • monthly, seasonal, annual mean anomalies
Levels:
  • Surface data.
Missing data: Grid cells for which insufficent data were available are flagged as missing in the data set..

Search keywords: precipitation, temperature, anomalies

Dataset format and size:

  • netCDF
  • Two monthly files, 20 Mbytes each
  • Two seasonal files, 6 Mbytes each
  • Two annual files, 2 Mbytes each
References:

P.D. Jones, S.C.B. Raper, B. Santer, B.S.G. Cherry, C. Goodess, P.M. Kelly, T.M.L. Wigley, R.S. Bradley, and H.F. Diaz (Eds.), 1985: A Grid Point Surface Air Temperature Data Set for the Northern Hemisphere. U.S. Dept. of Energy, Washington, DC, 251 pp.

J.K. Eischeid, H.F. Diaz, R.S. Bradley, and P.D. Jones (Eds.), 1991: A Comprehensive Precipitation Data Set for Global Land Areas. U.S. Dept. of Energy, Washington, DC, 82 pp.
